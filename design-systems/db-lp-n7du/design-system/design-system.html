Efeitos JS

Utilitários vanilla independentes. Cada arquivo funciona sozinho, sem dependências entre si.

smooth-scroll.js — Lenis smooth scroll

Inicializa o Lenis com easing exponencial (duration: 1.2s). O scroll nativo é substituído por um movimento inercial suave. Confirmado no CSS do original via classes .lenis.lenis-smooth.

initSmoothScroll(options?) // Retorna: instância Lenis (window.__lenis) // Depende: Lenis CDN
<script src="https://cdn.jsdelivr.net/npm/@studio-freight/lenis@1/bundled/lenis.min.js"></script> <script src="js/smooth-scroll.js"></script> <script>initSmoothScroll()</script>
Como funciona
scroll nativo
  ↓ interceptado pelo Lenis
easing exponencial 1.2s
  ↓ requestAnimationFrame loop
translateY suavizado
Role esta página para sentir o efeito ativo
parallax.js — Colunas com velocidades diferentes

.col-fast se move a 0.6× da velocidade de scroll e .col-slow a 0.3×, criando profundidade. Usa RAF throttling para performance. No card ao lado, mova o mouse para ver o efeito.

initParallax({ fastSelector: '.col-fast', fastSpeed: 0.6, slowSelector: '.col-slow', slowSpeed: 0.3 })
<div class="col-fast"><div class="card-image">...</div></div> <div class="col-slow"><div class="card-image">...</div></div> <script src="js/parallax.js"></script> <script>initParallax()</script>

nav-behavior.js — Active link + hide/show

Dois comportamentos: (1) IntersectionObserver detecta qual seção está visível e aplica .active no link correspondente. (2) Esconde a nav ao scrollar para baixo e reexibe ao subir.

initNavBehavior({ navSelector: '.nav', linkSelector: '.nav-link', hideThreshold: 100 })
<script src="js/nav-behavior.js"></script> <script>initNavBehavior({ navSelector: '.nav', linkSelector: '.nav-link' })</script>

Observa elementos com .reveal e adiciona .visible ao entrar na tela. Stagger automático entre irmãos (cada um atrasa staggerDelay × índice ms).
